Project #03 - Snow, Arrays, and Loops

Description:

Create an animation that depicts an animation of snow falling.

Requirements:

  1. Create a snowflake class that stores the properties of each individual snowflake.
  2. Your snowflake class should have a calculate and draw method.
  3. In your main class, create an array of snowflake objects. Then use a for-loop to loop through the array and store in each slot a newly created snowflake object.
  4. In your main class, inside your animation loop, loop through and call calculate on each snowflake. Somewhere later in your animation loop, loop through and call draw on each snowflake.
  5. The snow in your snowfall animation should fall indefinitely.
  6. Warning: do not copy and paste the example code we went over in class and try to replace what you need to. Learn from what we did in class, and use it to build your snowflake class and main class from the ground up.

Tips:

  • In order to make the snowflakes keep falling, make each snowflake "re-fall." In other words, when the snowflake drops below the screen, reset its position so that it is above the screen.
  • This is your first major project assignment. You need to build your code in small testable chunks.
  • Some examples of small testable steps:
    • Create your Main class and Snowflake class. In your snowflake class, create an xPos and yPos instance variable. Create a constructor that accepts no parameters and sets the values of this.xPos and this.yPos to 50. Create a draw method in your snowflake class that draws a circle at the xPos and yPos. In your main class, create one new snowflake object. In your animation loop, call the draw method on your snowflake. You should see a single stationary snowflake in the middle of the screen.
    • Add a calulate method to your snowflake class. Make a new instance variable for yVel. Set the default value for yVel in the constructor to 50. In your main method, call calulate on the snowflake in your animation loop. You should see a single stationary snowflake falling.
    • Make adjustments to your snowflake class so that the snowflake varies randomly in its starting position, velocity, size, etc. Test again to make sure your adjustments worked.
    • Make an empty array of 5 snowflake objects. Call the array snowflakes. Write a for-loop that cycles through the empty array. Create a new snowflake object at the start of your for-loop. Store the new snowflake object in the slot of the array that your for-loop is focused on for that iteration. Replace where you are currently drawing a single snowflake with a for-loop that cycles through your snowflakes array and draws the snowflake in each slot. Do the same thing for where you are currently calling calculate on a single object. Test to make sure your code works for all five snowflakes.
    • As you write your code, continue making adjustments in small, testable steps.

Grading

  1. Your code errors out while running or does not run at all. Or you do not meet one or more of the requirements stated in the requirements section. Or your snowfall animation does not resemble falling snow.
  2. Your snow falls in a straight line all at once.
  3. Your snowflakes fall with varying velocities (and thus at varying angles).
  4. Incorporate a few other unique variations in regard to the snowflakes themselves. Snowflakes could be varying sizes and shrink over time. You could use trigonometric functions to incorporate some oscillation in regard to the position or color of the snowflakes. These are just a few examples. Come up with other things on your own too! Have fun with it!
  5. Implement a complex feature in your main method that reacts in some way to the snowflakes or causes the snowflakes to behave in a different manner.
    • Example of a reaction to the snowflakes in the main method: You could make the snow "stick" to the ground. To do this, you could have your calculate method return a boolean value. Return false if the snowflake does not hit the ground in a given frame. Return true if it does hit the ground. In your main method react to this by increasing the level of the snow on the ground.
    • Example of something in the main method impacting the snowflakes: You could have lighter and darker clouds in your snowflake animation. If there is a darker cloud, you could have the snowflakes in that section of the screen fall faster and be larger. To accomplish this, you could have your calculate method accept a couple more parameters. Then from your main method you can pass in the location of the darker cloud. When you reset a snowflake to fall again, you can make the snowflake be larger if it falls from the darker cloud.
