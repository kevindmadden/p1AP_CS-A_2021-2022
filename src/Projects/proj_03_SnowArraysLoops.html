Project #03 - Snow, Arrays, and Loops

Description:

Improve the quality of the snow animation we went over in class. If you wish, you can also make something of your own from scratch as long as it meets the below requirements.

Requirements:

  1. Create at least two new arrays to use in making improvements to the snow animation.
  2. Use of for-loops to cycle through the elements in the new arrays you create.
  3. The improvement(s) you make must be meaningful.
  4. Examples of non-meaningful improvements:
    • You create a String array with a single element, perhaps to store the name of a color. You then use a for-loop that runs once to cycle through the one element in the array.
    • You change the color of the snowflakes to black.
  5. Examples of Meaningful Changes:
    • Create an array to store the y-velocity of each snowflake.
    • Create an array to store the x-velocity of the each snowflake. (This could be used to make snowflakes sway back and forth as they fall.
    • Create an array that sets the fall-angle of each snowflake.
    • Create an array that changes the size of the snowflakes in some manner over time.
    • Create arrays to store positions of clouds floating across the sky.
    • Create an array to store wind related affects on snowflakes.
